# Personal Portfolio

개인 포트폴리오 페이지입니다.  
HTML, CSS, JavaScript 기반으로 먼저 구현한 뒤,
**React**로 리팩토링된 최종 버전입니다.

<br>
<br>

## 🧩 Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript (ES6+)**
- **React**

<br>
<br>

## ✨ Features

- 반응형 레이아웃
- 메인 소개 섹션 (About)
- 프로젝트/작업물 리스트 섹션
- 간단한 연락/소셜 링크 섹션
- 재사용 가능한 기본 컴포넌트 스타일 (버튼, 카드 등)

<br>
<br>

## 🚀 How to Run

### 1. 이 레포지토리를 클론합니다.

```bash
git clone 레포지토리 주소
```

### 2. 새 터미널을 열고 개발자 모드로 실행합니다.

```bash
npm run dev
```
